# Orders API

1. Clone repo.
2. Run `yarn install`.
3. Clone `.env.template` and rename to `.env`.
4. Change the enviroment variables(if necessary).
5. Boot the DB

```
docker-compose up -d
```

6. Run SEED

```
http://localhost:3000/api/seed
```

7. Run: `yarn start:dev`.
